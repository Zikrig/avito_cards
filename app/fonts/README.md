# Шрифты для карточек

Чтобы **шрифты и текстовые блоки совпадали с примером** (example.svg), в эту папку нужно положить файлы шрифтов из макета:

- `MuseoSansVkusVill-100Italic.woff2` (или `.woff` / `.ttf`) — для описания и характеристик
- `MuseoSansVkusVill-900.woff2` (или `.woff` / `.ttf`) — для заголовка и цены

Имена файлов должны совпадать с указанными (без пробелов, с нужным регистром).

**Почему без шрифтов вид другой:**
- В шаблоне указаны шрифты MuseoSansVkusVill; в системе/браузере их нет.
- При экспорте в PNG используется запасной шрифт → другая ширина символов и высота строк → текст «разъезжается» и не совпадает с примером.
- В example.svg часть текста экспортирована из Illustrator как векторные контуры (path), поэтому там вид всегда одинаковый.

После добавления файлов в `app/fonts/` приложение будет подключать их при рендере, и карточка будет визуально совпадать с примером.
